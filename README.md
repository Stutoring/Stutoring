# Stutoring

Stutoring is a student-led tutoring service created by Jasper, Nikan, and Kiyan. We provide support in core academic subjects with a focus on clarity, patience, and real understanding.

## Subjects

- Math  
- Science  
- English  
- French  

## About

We built Stutoring to help students who feel stuck, confused, or behind in class. As students ourselves, we understand how topics are taught and where people usually struggle.

Our goal is simple: explain things in a way that actually makes sense.

## Website

This repository contains the official Stutoring website. It is a static site built using HTML.

## Features

- Clean and simple layout  
- Subject-based sections  
- Easy navigation  
- Informational content about tutoring services
